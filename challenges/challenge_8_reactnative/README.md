<h1 align="center">🚀 Eighth challenge</h1>

<blockquote align="center">Repository of Rocketseat's eighth GoStack Bootcamp challenge.</blockquote>

### ❓ What is it?
GoStack is a Bootcamp provided by Rocketseat where students learn to be Full-Stack developers using Node.js, React and React Native technologies. During the course students have to go through several challenges, which they have to complete in order to obtain the certificate.

### 🎓 Challenge
In the eighth challenge the student has to built a new mobile application named GoMarketplace. It can list the available products on the market, so that you can add a product to your card. It shows you the total card balance and the total products on the card.

### 🚧 Built with
- React Native
- Typescript

### 💡How to use
*First you'll have to use emulator to run the build, one as Android Studio can provide.*
- Once you already has the project open, run `yarn` to download dependencies.
- Run `yarn android/ios` to create a build.
